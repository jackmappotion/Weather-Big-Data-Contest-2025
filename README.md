# 2025 날씨 빅데이터 콘테스트

**KMA Weather Big Data Contest 2025**

> **수상:** 우수상 — 기상청장상 (Prize: 1,000,000 KRW)

---

## 📌 개요 (Overview)

* **주관:** 기상청(KMA)
* **기간:** 2025.05.07 ~ 2025.08.06
* **참여 형태:** 4인 팀
  
  * 백엔드 개발자
  * 임베디드 개발자
  * 기상 연구원
  * 데이터 분석가(본인)

### 🎯 목표

기상 변화와 대중교통 혼잡도 간의 상관관계를 규명하고, 이를 기반으로 **시간대별 혼잡도 예측 모델을 개발**하는 것이 목표.

---

## 🛠 프로젝트 수행 내용

### 1. 데이터 정제 (Data Cleaning)

* 코로나 시기(비정상적 혼잡 패턴)를 **전략적으로 제외**하여 모델의 예측 안정성을 확보.
* 이상치·결측치에 대해 도메인 기반 기준을 적용한 시계열 정제 수행.

### 2. 외부 데이터 수집 (External Data Enrichment)

* 혼잡도에 **지속적·정기적 영향을 주는 이벤트 데이터**를 직접 수집 및 구축.

  * 프로야구 경기 일정
  * 축구 리그 일정
  * 경기장 위치 → 인근 노선/정류장 매핑
* 기상 데이터 + 이벤트 데이터 + 교통 혼잡 데이터를 **하나의 예측용 시계열 테이블로 통합**.

### 3. 보고서 작성 및 발표

* 모델링 결과 해석 및 기상청 평가위원 대상 발표자료 제작.
* 팀 내 수집·정제·피처엔지니어링 관련 핵심 기술 파트 전담.

---

## 📈 주요 기술적 성과 (Key Contributions)

1. **외부 데이터 기반 가설검증 강화**

   * 도메인 기반 이벤트 변수 설계
   * 시계열 크로스코릴레이션을 활용한 영향도 검증

2. **데이터 융합 파이프라인 구축**

   * 기상·교통·이벤트 데이터를 통합한 Feature Table 제작
   * 팀에서 가장 많은 데이터 수집·정제·가공 업무를 수행

3. **예측 성능 향상에 기여**

   * 코로나 기간 제거 + 인과적 이벤트 변수 추가로 baseline 대비 성능 개선
